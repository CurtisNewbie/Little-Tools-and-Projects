
"CheckDirectorySize"

-----------------------------------------------------------
Descriptions:

This program checks the size of a folder through a number of recursive calls.

-----------------------------------------------------------
How to use it:

This program takes two arguments:

[0] the complete path to the directory/folder
[1] the mode (unit): -kb -mb -gb -tb

    "java CheckDirectorySize c:\users\curtis\someFolder -mb"


"PDF_To_Image"

-----------------------------------------------------------
Descriptions:

Convert the pdf file to a number of image using the "PDF Box
Library".

(Note: It does not modify original pdf file)

-----------------------------------------------------------
How to use it:

It is a Maven project, so Maven should be installed. It doesn't
take arguments. To run this program, type in the following command in the cmd:

"mvn clean compile exec:java"
